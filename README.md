# GitHub Copilot Demos

## Goal

This repository contains demo examples for **GitHub Copilot** courses. The demos are designed to showcase GitHub Copilot's key capabilities during live training sessions, covering topics such as code completions, chat-based refactoring, test generation, documentation, prompt engineering, bug detection, and code explanation.

---

## Project Structure

The repository is organized by programming language. Each language folder contains the same set of seven demos, each targeting a specific Copilot feature.

```
GitHubCopilot-Demos/
├── CSharp/       # C# demo examples
├── Java/         # Java demo examples
├── Javascript/   # JavaScript demo examples
└── Python/       # Python demo examples
```

| Folder | Language | README |
|--------|----------|--------|
| `CSharp/` | C# (.NET 8+) | [CSharp/README.md](CSharp/README.md) |
| `Java/` | Java (JDK 17+) | [Java/README.md](Java/README.md) |
| `Javascript/` | JavaScript (Node.js 18+) | [Javascript/README.md](Javascript/README.md) |
| `Python/` | Python 3.12+ | [Python/README.md](Python/README.md) |
