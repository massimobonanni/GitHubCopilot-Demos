This site hosts hands-on demo examples used in **GitHub Copilot** training courses.
Each demo targets a specific Copilot capability — code completions, chat-based
refactoring, test generation, documentation, prompt engineering, bug detection,
code explanation, plan mode, and Copilot customization (instructions, agents,
prompts, skills, and hooks).

## Language guides

The same set of demos is available in four languages. Pick your language to get
started:

| Language | Runtime | Guide |
|----------|---------|-------|
| **C#** | .NET 8+ | [C# demos](CSharp/README.html) |
| **Java** | JDK 17+ | [Java demos](Java/README.html) |
| **JavaScript** | Node.js 18+ | [JavaScript demos](Javascript/README.html) |
| **Python** | Python 3.12+ | [Python demos](Python/README.html) |
